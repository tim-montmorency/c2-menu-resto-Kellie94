## Description

Je suis allée avec un restaurant déjeuner inspiré par les robes d'été de ma mamie, c'est pourquoi j'ai choisi une image au motif fleuri et les couleurs qu'elle portait toujours: bleu, blanc et jaune. Le choix d'un fond foncé, plutôt que blanc est pour amener un côté sophistiqué et un semblant d'un mode sombre.

## Choix techniques

**Palette de couleurs :**

- Primaire : #183B65;
- Secondaire : #c6d0e0;
- #FEDE6F #00000088

**Police utilisée :**

- "Barlow", "Meow Script"

**Décisions CSS notables :**

- J'ai pris la liberté de mettre deux fonts pour bien séparer le titre du restaurant, les sections du reste du texte et les prix. La font "Meow Script" rapelle l'écriture en lettre attachée plus personnelle et caractérielle d'un personne et la font "barlow" pour la compréhension du reste du menu. J'ai décidé d'utiliser une image de font pour le main pour le séparer du font général de la page.
